Proyecto de Análisis de Calidad en Viñedo

Contexto Y datos Inicial

Este proyecto evalúa la calidad de la producción de uva en un viñedo con 14 parcelas, combinando registros históricos de clima, Calidad , pH. El objetivo final es optimizar decisiones de campo (riego, poda, control de plagas) y maximizar la calidad del vino (Merlot y Cabernet), Buscando contestar la pregunta planteada “Las tierras de mi viñedo podrán cubrir con la calidad al 100% de todas las uvas para tener un mejor producto al fin del año?” , para la realizacion de este proyecto se planteo una matriz de vision que tiene este proyecto.

Dataset bruto (captura en campo , Datos sinteticos)

Para iniciar el análisis, se desarrolló un conjunto de datos sintético en RStudio, diseñado para emular condiciones reales de producción en un viñedo con 14 parcelas, a lo largo de 5 años (2020–2024), con registros mensuales por cada etapa fenológica del ciclo de la vid: Cuajado, Envero y Meseta. La generación de datos sigue una lógica controlada de variación por parcela,año y etapa del ciclo. Este proyecto busca aplicar analisis de datos y crear un modelo predictivo que ayude a anticipar el rendimiento y calidad de dos tipos de vino: Merlot y Cabernet, optimizando la gestión del viñedo y reduciendo riesgos.

Para la elaboracion de los Datos de este proyecto se investigo que datos se podrian obtener en un viñedo real mediantes sensores tales como el pH.
